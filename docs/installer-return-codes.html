Chordia Installer Return Codes

This page documents return codes for the Chordia Windows installer.

Installer

File name: Chordia_3.0.0_x64-setup.exe
Installer type: NSIS setup executable
Silent install parameter: /S

Return codes

Return code Meaning Description
0 Installation successful The installation completed successfully.
1 Installation cancelled by user The installation was cancelled by the user.
2 Installation aborted by installer script The installation was aborted before completion. This may occur when the installer cannot complete the operation due to an unexpected installation failure.

Miscellaneous install failure scenarios

Chordia does not currently define separate custom return codes for application already exists, disk full, network failure, package rejection, or reboot required scenarios.
If such conditions cause the installer to stop, they are treated as a general installation abort and may return code 2.
