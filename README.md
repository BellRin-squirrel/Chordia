# 名称変更とリポジトリの過去について
このリポジトリは以前開発していたChordia_Windows（旧Chordia_Windows_Portable）とChordia_iOSの合併によるものです。
Chordia_WindowsはWindowsでもMacでも動作するように改めるため、Chordia_Desktopと名称を変更しました。（リリース名がChordia_WindowsやChordia_Windows_PortableのものはWindows版のみ対応、Chordia_Desktopの場合はWindows, Macに対応しています。）
Chordia_iOSはiOS, iPadOS, Androidのどれでも動作できるように改めるため、Chordia_Mobileと名称を変更しました。（リリース名がChordia_iOSやChordia_ipaのものはiOS, iPadOSのみ対応、Chordia_Mobileの場合はiOS, iPadOS, Androidに対応しています。）
バージョンはMobileとDesktopでのみ区別しており、iOS, iPadOS, Androidの対応についてそれぞれやWindows, Macの対応についてそれぞれ区別はしておりません。
例えば、バージョンは以下のように振られます。
Desktop版
Chordia_Windows v1.0.0
Chordia_Windows v1.0.1
Chordia_Desktop v1.1.0
Mobile版
Chordia_iOS v1.0.0
Chordia_iOS v1.0.1
Chordia_Mobile v1.1.0
Chordia_DesktopやChordia_Mobileをリリースした後にChordia_WindowsやChordia_iOSをリリースすることは原則ありません。
